About site

This site contains two games: Arduino 101 and Smart Eco City. The first one is a simple game, where you have to find some components like LED indicator, button ect. and combine them following simple steps to make a working device. The second one is a game, where you have to find garbage and recycle it. You also can send signals about contaminated places then will try to find volunteers to clean up the place.
